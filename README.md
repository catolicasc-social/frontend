## Usando localmente

#### Necessário

- [Node.js](https://nodejs.org/) (mais recente)
- [Yarn](https://yarnpkg.com/) (mais recente)

#### Comandos

- `git clone git@github.com:catolicasc-social/frontend.git`
- `cd src`
- `yarn install`
- `yarn dev`

Depois dos comandos executados, ir para http://localhost:3000
